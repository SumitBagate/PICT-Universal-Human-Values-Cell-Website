# PICT Universal Human Values Cell Website

This repository contains a responsive, single-page website for the Pune Institute of Computer Technology (PICT) Universal Human Values Cell.

## Overview

The site presents information about the UHV Cell, including its mission, vision, objectives, activities, committee members, gallery, and contact details. It is built as a static front-end using HTML, CSS, Bootstrap 5, Font Awesome, and vanilla JavaScript.

## Features

- Responsive navigation bar with smooth scrolling
- Hero section with call-to-action buttons
- Announcement marquee for updates
- About, mission, vision, objectives, and activities sections
- Committee, gallery, and contact sections
- Contact form validation on the client side
- Scroll-to-top button and scroll-based animations

## Project Structure

- `index.html` - main page structure and content
- `style.css` - custom styling and responsive layout rules
- `script.js` - smooth scrolling, navbar behavior, animations, and form handling
- `images/` - local images used by the site

## How To Run

1. Open `index.html` directly in a web browser.
2. Or use a local development server if you prefer, for example with VS Code Live Server.

## Notes

- The site loads Bootstrap and Font Awesome from CDNs, so an internet connection is required for those assets.
- The contact form is front-end only and currently shows validation messages without sending data to a backend.

## Customization

If you want to update the content, edit the relevant section in `index.html`. For visual changes, update `style.css`. For behavior changes such as scrolling or form handling, update `script.js`.
